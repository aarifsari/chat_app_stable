This is a simple real-time messaging application built with Flutter and Firebase
for the BIM493 Mobile Programming I course.

## Features

- Login screen with custom user ID
- Chat list screen with dummy users (user_0, user_1, ...)
- Real-time messaging between two users using Cloud Firestore
- Conversation-based message grouping (conversationId)
- Basic bubble-style chat UI

## Technologies

- Flutter
- Firebase Core
- Cloud Firestore

## How to run

1. Install Flutter SDK.
2. Run `flutter pub get`.
3. Connect a device or start an emulator.
4. Run:

```bash
flutter run
